# Expense Tracker Backend

A backend Expense Tracker built using Java, Spring Boot, Spring Data JPA, Hibernate, and MySQL.

This project is being developed from scratch as part of my backend engineering learning journey. The focus is on understanding backend architecture, clean code, REST API development, database design, authentication, and software engineering principles rather than simply completing a CRUD application.

---

## Tech Stack

* Java 21
* Spring Boot
* Spring Data JPA
* Hibernate
* MySQL
* Maven
* Eclipse IDE
* Postman

---

## Features (Planned)

### User Module

* User Registration
* User Login
* Password Encryption (BCrypt)
* JWT Authentication

### Expense Module

* Add Expense
* Update Expense
* Delete Expense
* View Expenses
* Filter by Category
* Filter by Month
* Monthly Expense Summary
